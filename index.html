JavaScript DirecTV Remote API by jcscoobyrs

The JavaScript DirecTV Remote API provides full access to the DirecTV set-top-box (STB) API allowing you to use JavaScript to control your STB. Right now, this project is in the very early stages but do not let this fool you as the API is complete and is completely unit tested.

The JavaScript DirecTV Remote API project is hosted on GitHub, and the annotated source code is available, as well as an online test suite and an example application.

You can report bugs and discuss features on the GitHub issues page or send tweets to @jcscoobyrs.

The JavaScript DirecTV Remote API project is in no way affiliated with or sanctioned by anyone at DirecTV and is solely the brainchild of jcscoobyrs.

The JavaScript DirecTV Remote API project has two hard dependencies:

  • jQuery 1.7 or greater
  • Underscore.js 1.2.2 or greater
